# gcloud & gsutil Cheat Sheet

## Storage (gsutil) Commands

### Create new bucket
`gsutil mb gs://<BUCKET_NAME>`

### Copy Uploaded File to Bucket
`gsutil cp [MY_FILE] gs://[BUCKET_NAME]`

## Compute Commands

### Show Regions List
`gcloud compute regions list`
